# LuaSTG 自机教程 [连载中]

欢迎来到 LuaSTG 自机教程！

我希望这个教程能帮助你了解 LuaSTG 的自机编写方法，制作出属于自己的自机。

我会尽量细致地进行讲解，但是我的能力有限，一些术语、语法等可能没有办法解释地很详尽，希望你能原谅。

这个教程基于 `LuaSTG aex+ v0.8.22` 版本，也许你看到这个教程时这个版本已经比较旧了，但我想这个教程仍然会有一定的参考价值。

阅读这个教程预计需要以下基础：
- 了解足够的 Lua 语法 (主要是 table 的用法)
- 能够使用自定义 obj 制作弹幕
- 有一个趁手的代码编辑器
- 时间与实践
