# LuaSTG 自机教程 [连载中]

欢迎来到 LuaSTG 自机教程！

本教程基于 `LuaSTG aex+ v0.8.22` 版本，如果你正在使用更旧或者更新的 LuaSTG，那么本教程的一些内容可能不适用于你的版本。但我想本教程仍然会有一些参考价值。

我会尝试尽量细致地进行讲解，但是我的时间和能力有限，一些术语、语法等可能没有办法解释得很详尽，希望读者谅解。

阅读这个教程预计需要以下基础：
- 了解足够的 Lua 语法 (主要是表和函数的用法)
- 了解面向对象的基本概念 (类、对象、继承等)
- 能够使用自定义 obj 制作弹幕
